# xcbox

Create temporary sandbox xcode project quickly.

# Examples

### new project 

install default is iOS

```
$ xcbox
```

### new iOS project

```
$ xcbox -i
```

### new macOS project

```
$ xcbox -m
```

# Management projects

open `$HOME/.xcbox`.

# Config

edit `$HOME/.xcbox/config.json`.
